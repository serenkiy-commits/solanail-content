# SoLaNail content repo

Source of truth for production site at solanail.ru.

- `live/` drives production site
- `draft/` reflects unsaved manager edits
- Edits via FastAPI admin at admin.solanail.ru (Olga) or directly via git push (Sergey)
- Templates only edited by Sergey via git

## Important rules

- Never edit `/var/www/solanail/*` directly — that's a build artifact
- Schema changes require migration script for existing products
- Photos in `media/products/` are originals; thumbs auto-generated by Pillow
- Videos in `media/products/videos/` are transcoded H.264 + AV1/WebM
